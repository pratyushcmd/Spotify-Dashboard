# Spotify Music Analytics Dashboard

## Overview
This is a personal data analytics and visualization project built using Power BI to explore trends in Spotify music data. The dashboard analyzes songs, artists, albums, popularity metrics, and release patterns over time. The project focuses on clean data modeling, meaningful DAX measures, and intuitive visual storytelling rather than just creating charts.

---

## Motivation
Music is something I interact with daily, and Spotify data offered a realistic dataset to practice end-to-end analytics. This project was built as a passion project to strengthen my skills in Power BI, DAX, and dashboard design while working with real-world-style data and answering practical analytical questions.

---

## Tech Stack
- Power BI  
- DAX (calculated measures and KPIs)  
- Spotify dataset (CSV)  
- Data modeling and relationships  

---

## Key Metrics
- 789 distinct songs  
- 342 unique artists  
- Average song duration: 3.28 minutes  
- Average popularity score: 90  

---

## Analysis Covered
- Songs by album type (album vs single)  
- Explicit vs non-explicit content distribution  
- Popularity trends by month and year  
- Top artists based on average popularity  
- Most popular songs  
- Monthly variation in distinct song releases  

---

## Dashboard Features
- KPI cards for quick insights  
- Artist and song-level popularity comparisons  
- Monthly and yearly trend analysis  
- Interactive slicers for time-based filtering  
- Spotify-inspired dark theme for better readability  

---

## How to Use
1. Open the `.pbix` file in Power BI Desktop  
2. Use slicers to filter by month or quarter  
3. Hover over visuals to view detailed tooltips  
4. Explore trends across artists, songs, and time  

---

## Learnings
- Writing optimized DAX measures for real use cases  
- Designing dashboards that balance clarity and aesthetics  
- Structuring data models for performance  
- Translating raw data into actionable insights
